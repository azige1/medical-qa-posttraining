# Results

把最终可展示的内容放在这里：

- `tables/`: 结果表格
- `figures/`: loss 曲线、对比图
- `case_studies/`: 典型成功和失败样例
- `predictions/`: 可公开的小规模预测样例
- `tensorboard/`: 不建议直接上传大文件，只保留说明或截图

建议最终至少沉淀以下结果：

- `Base / SFT / DPO / GRPO` 的总结果表
- `CSpider dev` 结果表
- 自建 SQLite 评测集结果表
- `execution_accuracy` / `valid_sql_rate` / `safe_sql_rate` 指标表
- `RM + RLOO/PPO mini` 小实验记录
- 至少 20 条错误案例归类
